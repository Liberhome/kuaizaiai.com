LifeRewind 隐私政策

生效日期：2026 年 5 月 6 日 · 最近更新：2026 年 5 月 6 日

快哉智能（澄迈）科技有限责任公司（以下简称"我们"）非常重视您的隐私。本隐私政策说明 LifeRewind iOS 应用（以下简称"本应用"或"LifeRewind"）如何收集、使用、存储、保护和披露您的信息。请您在使用本应用前完整阅读本政策。

一、概述

LifeRewind 是一款配合 Looki 智能设备使用的 iOS 应用，提供时间线回顾、AI 日报分析、漫画生成、人生预言等功能。本应用主要在您的设备本地以及您指定的 iCloud 账户中处理数据。我们尽可能减少向第三方服务传输的信息量，并使用业内通行的加密手段。

二、我们收集的信息

2.1 您主动提供的信息

  • API 密钥配置（v1 仅个人开发模式）：您可在应用设置中粘贴自有的火山引擎、阿里云、DeepSeek、OpenAI、Anthropic 等服务的 API Key。这些密钥仅以 iOS Keychain 加密形式存储在本地设备，不会上传到我们的服务器。
  • Looki 账号凭据：用于从您的 Looki 智能设备同步生活片段。凭据由 Looki 官方机制管理。
  • 预言家目标设定：您输入的人生目标文本，仅在生成预言时通过指定大模型 API 进行处理。

2.2 通过设备生成或同步的内容

  • 生活片段（moments）：从 Looki 设备同步的时间、地点、活动、照片、视频等。
  • AI 生成内容：日报、漫画、预言结果。生成结果存储在本地及您的 iCloud 容器中。
  • 使用配额数据：当月已使用的预言/漫画次数、点数包余额，仅存储于本地 UserDefaults，用于配额管理，不上传。

2.3 与购买相关的信息

  • App Store 订阅与内购凭证：通过 Apple StoreKit 处理，由 Apple 直接管理。我们仅接收订阅状态和内购成功标志，不接触您的支付方式、信用卡号、姓名或邮箱。

2.4 我们不收集的信息

我们不收集您的设备 IDFA / 广告标识符、不进行行为追踪、不接入第三方分析 SDK（如友盟、Firebase Analytics、Sensors 等）、不在后台静默上传您的相册或通讯录、不读取您的位置信息（除非您主动在 moment 中含有位置标签）。

三、信息使用目的

  • 为您提供本应用的核心功能（时间线、日报、漫画、预言、回忆墙）。
  • 记录您的订阅状态与配额使用情况，用于功能授权。
  • 响应您的支持请求与反馈。
  • 遵守适用法律法规要求。

我们不会将您的信息用于上述目的之外的用途，不会用于广告投放、不会出售给任何第三方。

四、第三方服务（API 调用清单）

为提供 AI 功能，本应用会在您主动触发对应功能时，将必要的提示词和参数发送给以下第三方服务。我们仅传输完成功能所必需的最少内容，且不在请求中附带可识别您身份的元数据：

  • 火山引擎 Seedream（字节跳动）：将漫画生成提示词发送给图像生成 API。隐私政策。
  • 阿里云通义万相 Qwen-VL：可选用于图像识别（视觉标签）。阿里云政策。
  • DeepSeek：将预言提示词发送给对话模型 API。DeepSeek 隐私政策。
  • Anthropic Claude：可选预言模型。Anthropic 隐私政策。
  • OpenAI ChatGPT：可选预言模型。OpenAI 隐私政策。
  • Looki Tech：智能项链生活片段同步。Looki 政策。
  • Apple StoreKit / iCloud：支付与跨设备数据同步，由 Apple 管理。Apple 隐私政策。

跨境传输提示：上述部分服务的服务器可能位于中国大陆境外。一旦您选择使用对应模型，您的提示词将依据相应服务商的隐私政策进行处理。

五、数据存储与保留

  • 本地数据：除生成内容、配额、API 密钥外，绝大部分数据均存储于您设备本地。
  • iCloud 数据：通过 Apple CloudKit 在您的 iCloud 账户内同步，存储于 Apple 数据中心，端到端加密由 Apple 管理。
  • 保留期限：您可以随时在应用内删除生成内容；卸载应用时，本地数据全部清除；iCloud 数据可在 Apple ID 设置中按 LifeRewind 容器单独清除。

六、数据安全

所有 API 网络通信均使用 HTTPS / TLS 1.2+ 加密。本地敏感数据（如 API 密钥）存储于 iOS Keychain，受系统级加密保护。我们不在服务器端保留您的内容副本（v1 阶段无自有后端）。

七、儿童隐私

本应用不面向 13 周岁以下儿童，且不会主动收集儿童个人信息。如您是未成年人，请在监护人陪同下使用本应用。如您发现儿童在未经授权情况下提供了个人信息，请通过下方邮箱联系我们立即删除。

八、您的权利

  • 访问与导出：本应用内所有由您产生的内容均可在应用内查看；漫画/预言结果可保存到相册或导出。
  • 删除：您可在应用内逐条删除内容；卸载应用即清除全部本地数据。
  • 取消订阅：通过 iOS"设置 → Apple ID → 订阅"管理。
  • 申诉与查询：通过 liber@kuaizaiai.com 联系我们处理任何隐私相关请求。我们将在 15 个工作日内回复。

九、政策更新

我们可能不定期更新本政策。重大变更会在应用内显著位置或通过推送通知告知您。继续使用即视为接受更新后的政策。
